## Tech Stack
This website is built with:
- **Framework:** Next.js 15.3.2
- **Language:** TypeScript
- **UI Components:** shadcn-ui
- **Styling:** Tailwind CSS
- **Form Handling:** react-hook-form
- **Theme:** next-themes for dark/light mode
- **Validation:** zod
